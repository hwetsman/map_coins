This will be an attempt to map ancient Judean coins over time by horizon
limits.
Currently we envision these chronological types:
1. Hasmonean coins
    A. John Hyrcanus I
    B. Alexander Jannaeus
    C. Matityahu Antigonus
2. Herod and his descendants
    A. Herod I
    B. Herod Archelaus
    C. Herod Antipas
    D. Herod Philip II
    E. Agrippa I
    F. Agrippa II
3. First Revolt
4. Second Revolt
